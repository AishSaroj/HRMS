# HRMS Lite - Full Stack HR Management System

A complete HR Management System built with React frontend and RESTful API backend. This application allows administrators to manage employees and track their daily attendance.

## Features

### Core Features
1. **Employee Management**
   - Add new employees with validation
   - View all employees in a searchable table
   - Delete employees
   - Department-wise filtering

2. **Attendance Management**
   - Mark daily attendance for employees
   - Filter attendance by employee and date
   - View attendance statistics
   - Delete attendance records

3. **Dashboard**
   - Real-time statistics
   - Employee attendance summary
   - Department-wise employee count

### Technical Features
- Responsive design
- Form validation
- Loading states
- Error handling
- Filtering and search
- Inline CSS styling

## Tech Stack

### Frontend
- **React** - UI Framework
- **Inline CSS** - Component-based styling
- **JavaScript ES6+**

### Backend (To be implemented)
